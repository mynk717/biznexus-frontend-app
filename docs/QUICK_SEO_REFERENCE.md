# ⚡ Quick SEO Reference (30-Second Checklist)

**Use this before starting ANY article work.**

---

## 🎯 The 10 Non-Negotiables

### 1. E-E-A-T Signals
- [ ] Personal/local experience mentioned
- [ ] Expert credentials shown
- [ ] Brand authority established
- [ ] Facts verified & sources cited

### 2. Primary Keyword
- [ ] In title (first 5 words)
- [ ] In H1 (naturally)
- [ ] In first 100 words
- [ ] In URL slug
- [ ] In meta description

### 3. Content Structure
- [ ] H1 (only ONE)
- [ ] H2s (4-8) with secondary keywords
- [ ] H3s with long-tail keywords
- [ ] Short paragraphs (2-4 sentences)
- [ ] Bullet points/lists used

### 4. Schema Markup
- [ ] Article/BlogPosting (always)
- [ ] FAQPage (10-15 Q&As)
- [ ] Product (if applicable)
- [ ] HowTo (if guide)
- [ ] Validated in Rich Results Test

### 5. "People Also Ask"
- [ ] 10-15 PAA questions captured
- [ ] Each answered in H2/H3
- [ ] Answers concise (40-60 words)

### 6. Internal Links
- [ ] 3-5 contextual links
- [ ] Descriptive anchor text
- [ ] Related pillar/cluster content
- [ ] CTA to service/contact page

### 7. External Links
- [ ] 2-3 authority links (.gov/.edu)
- [ ] pmsuryaghar.gov.in (if solar)
- [ ] creda.cgstate.gov.in (if Raipur)
- [ ] Industry authorities

### 8. Local SEO (Raipur)
- [ ] "Raipur" in title, H1, first para
- [ ] "Chhattisgarh" mentioned
- [ ] CSPDCL/CREDA mentioned
- [ ] Local context throughout

### 9. Technical
- [ ] Page speed <3s
- [ ] Mobile responsive
- [ ] All images <100KB (WebP)
- [ ] Alt text on all images
- [ ] No broken links

### 10. CTAs
- [ ] Primary CTA above fold
- [ ] CTAs every 800-1000 words
- [ ] Action-oriented text
- [ ] Stands out visually

## 📊 Quick Metrics Targets

| Metric | Target |
|--------|--------|
| Word Count | 2,500-4,000 (pillar) |
| Keyword Density | 1-2% (primary) |
| Reading Level | Grade 8-10 |
| H2 Sections | 4-8 |
| Internal Links | 3-5 |
| External Links | 2-3 |
| FAQs | 10-15 |
| Images | 3-5 |
| Page Speed | <3s |

---

## 🚨 Red Flags (Stop & Fix)

- ❌ No E-E-A-T signals (experience, expertise, trust)
- ❌ Keyword stuffing (reads unnaturally)
- ❌ No schema markup
- ❌ No internal links
- ❌ Generic content (not Raipur-specific)
- ❌ No "People Also Ask" coverage
- ❌ Missing meta description
- ❌ Duplicate H1s
- ❌ Images >200KB
- ❌ No mobile test
